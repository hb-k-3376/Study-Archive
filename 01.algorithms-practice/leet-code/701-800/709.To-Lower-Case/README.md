# LeetCode 709. To Lower Case

## 문제 설명

문자열을 소문자로 변환하는 문제입니다.

문자열 s가 주어졌을 때, 문자열을 소문자로 변환하여 반환해야 합니다.

### 예제

**Example 1:**

- Input: s = "Hello"
- Output: "hello"

**Example 2:**

- Input: s = "here"
- Output: "here"

**Example 3:**

- Input: s = "LOVELY"
- Output: "lovely"

## 제한사항

- 1 <= s.length <= 100
- s는 출력 가능한 ASCII 문자로만 구성됨
